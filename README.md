# 드래그 앤 드롭 라이브러리

## 예제코드
### HTML
```html
<div id="workspace">
</div>
```
### JS
```js
new Drag("workspace");
```

## 함수

### 블록
- addBlock : 드래그 후 블록을 추가합니다.
- removeBlock : 나오는 X 버튼을 클릭하여 블록을 제거합니다.

### 연결선
- addLine : 블록의 이음새부분을 클릭시 라인이 생성됩니다.
- removeLine : 블록의 이음새부분을 클릭 빈공간을 클릭하면 연결선이 사라집니다.
